# Простое веб-приложение с формой

Одностраничное веб-приложение с формой обратной связи. Данные сохраняются в файл на сервере.

## Установка и запуск

### 1. Установите зависимости

```bash
npm install
```

### 2. Запустите сервер

```bash
npm start
```

Сервер запустится на `http://localhost:3000`

### 3. Откройте приложение в браузере

Перейдите по адресу: http://localhost:3000

## Структура проекта

- `index.html` - главная страница с формой
- `style.css` - стили приложения
- `script.js` - клиентская логика
- `server.js` - сервер на Node.js/Express
- `package.json` - зависимости проекта
- `data/submissions.json` - файл для хранения данных (создается автоматически)

## Функциональность

- Форма с тремя полями: имя, почта, комментарий
- Валидация полей
- Сохранение данных в файл `data/submissions.json`
- Состояние успеха после отправки
- Красивый современный интерфейс

## Просмотр сохраненных данных

Все данные сохраняются в файл `data/submissions.json` в формате JSON.

Также можно получить данные через API:
- GET запрос на `http://localhost:3000/api/submissions`

Или в консоли браузера:
```javascript
getAllSubmissions()
```
